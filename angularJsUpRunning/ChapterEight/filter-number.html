Filters Application Using AngularJS


Numbers:
  • Amout without any filter: {{ ctrl.amount }}
  • Amount using default currency: {{ ctrl.amount | currency }}
  • Amount using currency: {{ ctrl.amount | currency:'£'}}
  • Amount using number: {{ ctrl.amount | number }}
  • Amount using number 4 decimal: {{ ctrl.amount | number: 4}}
String name:
  • Name without any filter: {{ ctrl.name }}
  • Name lowercase filter: {{ ctrl.name | lowercase }}
  • Name uppercase filter: {{ ctrl.name | uppercase }}
JSON
  • Without filter: {{ ctrl.obj}}
  • The JSON Filter: {{ ctrl.obj | json }}
Time & Day
  • Timestamp or without filter: {{ ctrl.startTime }}
  • Default time filter: {{ ctrl.startTime | date }}
  • Medium Date Filter: {{ ctrl.startTime | date:'medium' }}
  • Custom Date Filter: {{ ctrl.startTime | date:'M/dd, yyyy' }}
  • Custom Date Filter: {{ ctrl.startTime | date:'yyyy/dd/M' }}
